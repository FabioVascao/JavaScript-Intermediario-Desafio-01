# JavaScript-Intermediario-Desafio-01
 
Desafio do Programa Explorer solicitando um timer com contagem regressiva e com 4 opções de músicas. Cada card que se encontra do lado direito, quando selecionado, vai mudar de cor e emitir um som diferente. 

Funcionalidades esperadas dos botões:

- Play   : aciona o timer;
- Stop   : para o timer;
- +    : aumenta em mais 5 minutos o tempo do timer;
- -    : diminui em 5 minutos o tempo do timer.

Tecnologias utilizadas: HTML - CSS -  JavaScript - Git e Github.
